记录我的第一篇博客 by彭月明

记录我的第一篇博客 by Vlaq

折腾了一上午，从找教程、下软件、配置环境、到现在写下这篇随笔，算是简单的知道了怎么撰写、发布博文在自己的博客上了。当然，我现在肯定只能使用一些非常简单的技术来搭建博客，教程也几乎是保姆式的，一步一步跟着做就好了。现在面临一个新的问题，撰写文章要用到一种叫MarkDown的文本标记语言，没接触过过这东西，得花点时间钻研下。现在我正在使用在线Markdown编辑器dillinger写下这篇文章。

简单记录一下搭建个人博客的过程，主要参考了以下两篇教程：
1.hexo+github搭建个人博客(超详细教程)
https://blog.csdn.net/AinUser/article/details/77609180?utm_medium=distribute.pc_relevant_t0.none-task-blog-BlogCommendFromMachineLearnPai2-1.nonecase&depth_1-utm_source=distribute.pc_relevant_t0.none-task-blog-BlogCommendFromMachineLearnPai2-1.nonecase
2.使用 Hexo+GitHub 搭建个人免费博客教程（小白向）
https://zhuanlan.zhihu.com/p/60578464

  • 下载Git、Node.js两款软件
  • 创建GitHub仓库，设置相关参数
  • 配置本地环境
  • 部署 Hexo 到 GitHub Pages
  • 撰写文章
    ps:暂时没买域名，所以没有绑定个人域名这一步操作。

暂时就写到这里啦，这个博客主要是用于记录我学习编程时的一些笔记、感想之类的，也算是对我的学习经历的一个记录。
